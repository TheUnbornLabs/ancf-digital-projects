# Economic Pressure Explainer

*Project 030 · Category: Society · Difficulty: Intermediate*

How economics shapes family decisions in both directions. Five "forces" cards (housing,
childcare/education, work/income, old-age security, debt); a **two-direction pressure mapper**
(rate forces pushing toward vs away → a net-pull needle and reading that maps pressure, not a
verdict); and a "structures, not blame" section. Vanilla JS; DOM test 7/7.
