# Instagram Carousel Text Builder

*Project 074 · Category: Writing Tools · Difficulty: Beginner*

Builds a five-slide educational carousel (hook / three points / kind close) across five topics
(pronatalism, childfree, autonomy, pressure, antinatalism basics) and three tones, rendered as
slide cards with per-slide copy and a copy-all. Respectful and accurate. Vanilla JS; DOM test 7/7.
