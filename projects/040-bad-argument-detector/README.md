# Bad Argument Detector

*Project 040 · Category: Learning Tools · Difficulty: Intermediate*

Scans pasted text for fifteen common logical fallacies (regex-pattern heuristics), flagging the
matched phrase with a plain explanation and an honest "use your judgement" framing. Three sample
arguments, plus a filterable/searchable field guide of all fifteen fallacies with examples.
Vanilla JS; DOM test 8/8.
