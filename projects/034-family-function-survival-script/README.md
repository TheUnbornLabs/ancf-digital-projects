# Family Function Survival Script

*Project 034 · Category: Writing Tools · Difficulty: Beginner*

Builds a four-line pocket script (opener / deflection / firm line / graceful exit) tuned to the
event and your style, with a shuffle and copyable card. Includes a tap-to-reveal field guide of
likely lines with ready responses, and a saved readiness checklist with progress. Vanilla JS;
DOM test 8/8.
