# Antinatalist Statement Builder

*Project 073 · Category: Writing Tools · Difficulty: Intermediate*

Assembles a structured antinatalist position statement from a basis (asymmetry / consent / risk /
suffering-focused / environmental), a tone, and a length — claim, reasoning, and an explicit "this is
ethics, not a judgement of parents" caveat. Includes a respect-check list and a guard that warns if
demeaning language appears in an edit. Vanilla JS; DOM test 6/6.
