Admin Warning Message Generator

Community Tools · Difficulty: Intermediate

A good moderation warning is firm, fair, and private. It names the behaviour, never the person's worth, and it gives a clear path forward. Pick an issue and step to draft one, then adapt it to your community's voice.

1. Generate a warning

Your warning message will appear here.

2. Make it yours

3. De-escalation tips

Send it privately

Public call-outs shame and harden people. A private DM lets someone correct course without losing face.

Name the behaviour, not the person

"This message broke rule 1" works; "you're a bad person" doesn't. Keep it about the action and the rule.

Be clear about consequences

State plainly what happens next if it continues — predictability is fairer than surprise removals.

For self-harm, lead with care

If someone expresses thoughts of self-harm, this is not a discipline issue. Respond with warmth, share support resources, and never warn or punish them for it.
