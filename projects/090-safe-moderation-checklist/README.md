# Safe Moderation Checklist

*Project 090 · Category: Community Tools · Difficulty: Intermediate*

A four-group, 20-item moderation checklist (setting up, day-to-day, when things go wrong, and a
highlighted moderator self-care group), with per-group counts, an overall progress meter, a copyable
checklist export, reset, and three principle cards. Saved on-device. Vanilla JS; DOM test 6/6.
