PROJECT 015

Life Goals Priority Mapper

Writing Tools · Difficulty: Beginner

Bring your goals into focus by sorting them across three time horizons. A childfree or examined life has just as many goals as any other — this tool helps you name and place them. Everything stays on this device.

How to use it

Write one goal per line under each horizon. Move an item by cutting it from one box and pasting it into another. The counts update as you type. Need a spark? Tap a starter idea below and it drops into "Now" for you to edit or move.

Starter ideas:

Now (this year)

0 items

Soon (1–3 years)

0 items

Someday / maybe

0 items

Saved on this device only.

Tip: "Someday/maybe" is not a lesser column — parking a goal there frees you to focus, without losing it.
