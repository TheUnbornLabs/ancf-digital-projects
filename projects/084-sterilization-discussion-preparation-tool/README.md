# Sterilization Discussion Preparation Tool

*Project 084 · Category: Childfree Life · Difficulty: Intermediate*

A topic-toggle question-list builder (options, permanence, risks, recovery, alternatives, consent/
autonomy, access) that compiles tailored questions to ask a provider, with copy and select-all, plus
four "worth thinking through first" cards. Strictly non-medical — no procedure detail or recommendation,
just question organisation. Vanilla JS; DOM test 7/7.
