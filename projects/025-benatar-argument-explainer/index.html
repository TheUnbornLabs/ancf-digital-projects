PROJECT 025

Benatar Argument Explainer

Philosophy · Difficulty: Advanced

This explains a famous academic argument in accessible terms, then gives the strongest objections, so you can judge for yourself.

The asymmetry

The argument claims that the absence of pain is good (even if no one enjoys that good), while the absence of pleasure is not bad unless someone is deprived of it. From this, some conclude coming into existence is always a harm.

Why some find it powerful

It tries to explain a common intuition: we pity a suffering life, but we don’t mourn lives that were never created.

The main objections

Critics argue the asymmetry is question-begging, that 'good for no one' is incoherent, and that lives can be net positive. Many philosophers reject the conclusion.

Reflection prompt

Which step in the argument do you find weakest, and why?
