# Nietzsche Misuse Explainer

*Project 059 · Category: Philosophy · Difficulty: Advanced*

Explains three engines of Nietzsche distortion (flattening, his sister's editing, reading backwards),
a misquote decoder (six lines: popular misreading vs closer reading, with sources), and an "is this
really Nietzsche?" classifier (genuine / distorted / misattributed) with feedback. Carefully sourced.
Vanilla JS; DOM test 7/7.
