# Rationalism Toolkit

*Project 060 · Category: Learning Tools · Difficulty: Intermediate*

Seven thinking-tool cards (steelman, separate fact/value/definition, what-would-change-my-mind,
seek disconfirmation, taboo the word, check the base rate, notice motivated reasoning); an
"apply a tool to a claim" generator that produces tailored guiding questions (with samples); and a
seven-item self-audit that names your least-used tool. Vanilla JS; DOM test 9/9.
