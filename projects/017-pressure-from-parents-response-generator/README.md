# Pressure-from-Parents Response Generator

*Project 017 · Category: Writing Tools · Difficulty: Beginner*

Drafts replies to parental pressure that honor love while holding the line. A warmth-to-firmness
dial + situation + goal generate an acknowledge-then-hold reply (with variants); an "underlying
worry" mapper decodes the fear beneath common lines and offers a sentence that speaks to it.
Distinct from the boundary tools by its parent-specific, empathy-first framing. Vanilla JS;
DOM test 7/7.
