# Gaslighting Phrase Detector

*Project 043 · Category: Society · Difficulty: Intermediate*

Scans a pasted message for six gaslighting techniques (denial, memory-doubt, trivializing,
discrediting, blame-shifting, withholding), each flagged with what it does, the matched phrase,
and a reality anchor. Includes a technique guide, five grounding practices, three samples, and a
non-diagnostic support note. Vanilla JS; DOM test 7/7.
