# "When Are You Having Kids?" Reply Tool

*Project 035 · Category: Community Tools · Difficulty: Beginner*

Generates polite one-liner replies to the most common question, by vibe (gracious/breezy/honest/
firm/deflect) and who's asking, with cycling and copy. Lets you assemble your three go-to lines
into a saved, copyable kit (capped at three). Boundary, never insult. Vanilla JS; DOM test 6/6.
