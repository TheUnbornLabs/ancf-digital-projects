# Feminist Childfree Explainer

*Project 029 · Category: Society · Difficulty: Intermediate*

How feminism connects to the childfree choice, mapped honestly. Five interactive "threads"
(bodily autonomy, beyond destiny, equality at work, the unequal load, solidarity); an expandable
"voices & milestones" list (de Beauvoir 1949 → Donath 2017, dates verified); and three real
debates *within* feminism (choice vs structural critique, devaluing motherhood, whose feminism)
with fair responses. Defends no single line. Vanilla JS; DOM test 6/6.
