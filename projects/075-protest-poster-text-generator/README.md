# Protest Poster Text Generator

*Project 075 · Category: Writing Tools · Difficulty: Beginner*

Generates positive, rights-based slogans across five focuses (autonomy, choice, dignity, education,
solidarity) and three styles (plain/rhyme/gentle), rendered on a poster card, with cycling, copy,
and saved favourites. Affirming, never anti-anyone. Vanilla JS; DOM test 5/5.
