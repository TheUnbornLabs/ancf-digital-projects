# Future Without Forced Parenthood — Vision Board

*Project 099 · Category: Writing Tools · Difficulty: Beginner*

A hopeful, single-user vision board: 12 tappable prebuilt "sparks" plus a free-text add, rendered as
removable tiles, with copy/export and clear. Saved on-device. Pictures a world of free, supported
reproductive choice. Vanilla JS; DOM test 5/5.
