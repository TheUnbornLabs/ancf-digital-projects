PROJECT 038

Admin Warning Message Generator

Community Tools · Difficulty: Intermediate · ~7 min

Moderation works best when it's firm and fair. This tool drafts a private warning that names the behaviour, not the person's worth, and offers a way back. Pick the issue and the step, edit, and send privately.

For fair moderation. Address behaviour, never a person's identity. Keep warnings private; protect targets of harassment first. Not legal advice.

What you will learn

  • How to name a behaviour without attacking the person.
  • How warnings escalate fairly: reminder → formal → final.
  • How to leave a clear path back to good standing.

Key takeaway

Fair moderation is specific, private, and proportionate: name the behaviour, cite the rule, and leave a clear path back. Protect targets first — kindness to the room sometimes means firmness with one person.
