# Respectful Language Guide

*Project 089 · Category: Community Tools · Difficulty: Beginner*

A filterable/searchable "say this, not that" library of 14 swaps across five contexts (about parents,
the childfree, in debate, about groups, self-talk), each with instead-of / try / why and a copy of the
kinder version. Aims at ideas not people; no group-targeting. Vanilla JS; DOM test 6/6.
