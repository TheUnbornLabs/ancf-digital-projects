PROJECT 019

Childfree Couple Planning Board

Childfree Life · Difficulty: Beginner

A childfree partnership is not an absence of plans — it's a different set of them. Use this board together to name what you're building across four areas. Everything stays on this device; share the screen, not your data.

How to use it

Sit down together and add one item per line in each column. Tap a starter prompt to drop an idea into its column, then make it your own. Counts update as you type.

Starter prompts:

Shared dreams

0 items

Time & travel

0 items

Care network (chosen family)

0 items

Saved on this device only.

The "care network" column matters: childfree people build strong webs of chosen family and friendship. Planning that on purpose is one of the kindest things a couple can do.

This project is for education and reflection only. It is not medical, legal, financial, or mental health advice.
