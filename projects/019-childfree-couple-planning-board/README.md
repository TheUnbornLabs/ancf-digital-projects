# Childfree Couple Planning Board

*Project 019 · Category: Childfree Life · Difficulty: Beginner*

An editable vision board across six life areas (travel, home, growth, people, health,
contribution). Add your own items or tap a spark; remove anything; build a shared-vision
statement from two prompts; export the whole board. All on-device. Vanilla JS; DOM test 7/7.
