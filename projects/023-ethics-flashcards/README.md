# Ethics Flashcards

*Project 023 · Category: Learning Tools · Difficulty: Beginner*

A 24-card study deck for birth-ethics vocabulary. Flip cards, mark "known", filter by five
categories, and watch a progress bar fill. A focused **study session** mode shuffles the deck (or
just the unmastered cards), tests recall one card at a time, and scores you. All on-device.
Vanilla JS; DOM test 9/9.
